# Assebmler Vzlom (Dos hacking)
## Описание
Программа запрашивает у пользователя пароль. Если пользователь ввел верный пароль, выдается доступ (access granted), иначе - нет (acess denied).

Пример:
<img width="1008" height="360" alt="image" src="https://github.com/user-attachments/assets/2bb10f21-d99f-4ebe-a7df-49fa7283e245" />

От друга я получила .com файл с его программой. В ней заложено две уязвимости. Моя задача - найти эти уязвимости и получить доступ к программе, не зная при этом пароля.
В ходе обстоятельств, я напишу про взлом двух разных программ, в каждой из которых опишу по одной из яузвимостей.

Для взлома будет использоваться встроенная функция dosbox - turbo debugger.
